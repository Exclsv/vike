<ImplementedBy by={<code>vike-react</code>} noCustomGuide={true}>the `Loading` setting</ImplementedBy>

The the `Loading` setting helps you create page/layout level Loading UI.<br/>
With this config, you can show an instant loading state while the content of a route segment loads.<br/>
The new content is automatically swapped in once rendering is complete.


```jsx
// pages/+Loading.jsx

export default function Loading() {
  // Applies to the page and all layouts
  return <div>Loading...</div>
}
```
